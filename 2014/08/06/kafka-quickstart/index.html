Kafkaеҝ«йҖҹе…Ҙй—Ё

зј–иҜ‘иҮӘе®ҳж–№ж–ҮжЎЈ

Apache KafkaжҳҜдёҖдёӘеҲҶеёғејҸж¶ҲжҒҜеҸ‘еёғи®ўйҳ…зі»з»ҹгҖӮе®ғжңҖеҲқз”ұLinkedInе…¬еҸёеҹәдәҺзӢ¬зү№зҡ„и®ҫи®Ўе®һзҺ°дёәдёҖдёӘеҲҶеёғејҸзҡ„жҸҗдәӨж—Ҙеҝ—зі»з»ҹ( a distributed commit log)пјҢпјҢд№ӢеҗҺжҲҗдёәApacheйЎ№зӣ®зҡ„дёҖйғЁеҲҶгҖӮKafkaзі»з»ҹеҝ«йҖҹгҖҒеҸҜжү©еұ•е№¶дё”еҸҜжҢҒд№…еҢ–гҖӮе®ғзҡ„еҲҶеҢәзү№жҖ§пјҢеҸҜеӨҚеҲ¶е’ҢеҸҜе®№й”ҷйғҪжҳҜе…¶дёҚй”ҷзҡ„зү№жҖ§гҖӮ

Apache KafkaдёҺдј з»ҹж¶ҲжҒҜзі»з»ҹзӣёжҜ”пјҢжңүд»ҘдёӢдёҚеҗҢпјҡ

  • е®ғиў«и®ҫи®ЎдёәдёҖдёӘеҲҶеёғејҸзі»з»ҹпјҢжҳ“дәҺеҗ‘еӨ–жү©еұ•пјӣ
  • е®ғеҗҢж—¶дёәеҸ‘еёғе’Ңи®ўйҳ…жҸҗдҫӣй«ҳеҗһеҗҗйҮҸпјӣ
  • е®ғж”ҜжҢҒеӨҡи®ўйҳ…иҖ…пјҢеҪ“еӨұиҙҘж—¶иғҪиҮӘеҠЁе№іиЎЎж¶Ҳиҙ№иҖ…пјӣ
  • е®ғе°Ҷж¶ҲжҒҜжҢҒд№…еҢ–еҲ°зЈҒзӣҳпјҢеӣ жӯӨеҸҜз”ЁдәҺжү№йҮҸж¶Ҳиҙ№пјҢдҫӢеҰӮETLпјҢд»ҘеҸҠе®һж—¶еә”з”ЁзЁӢеәҸгҖӮ

First let's review some basic messaging terminology:
йҰ–е…ҲжқҘдәҶи§ЈдёҖдёӢKafkaжүҖдҪҝз”Ёзҡ„еҹәжң¬жңҜиҜӯпјҡ

Topic
Kafkaе°Ҷж¶ҲжҒҜз§Қеӯҗ(Feed)еҲҶй—ЁеҲ«зұ»пјҢ жҜҸдёҖзұ»зҡ„ж¶ҲжҒҜз§°д№ӢдёәиҜқйўҳ(Topic).
Producer
еҸ‘еёғж¶ҲжҒҜзҡ„еҜ№иұЎз§°д№ӢдёәиҜқйўҳз”ҹдә§иҖ…(Kafka topic producer)
Consumer
и®ўйҳ…ж¶ҲжҒҜе№¶еӨ„зҗҶеҸ‘еёғзҡ„ж¶ҲжҒҜзҡ„з§Қеӯҗзҡ„еҜ№иұЎз§°д№ӢдёәиҜқйўҳж¶Ҳиҙ№иҖ…(consumers)
Broker
е·ІеҸ‘еёғзҡ„ж¶ҲжҒҜдҝқеӯҳеңЁдёҖз»„жңҚеҠЎеҷЁдёӯпјҢз§°д№ӢдёәKafkaйӣҶзҫӨгҖӮйӣҶзҫӨдёӯзҡ„жҜҸдёҖдёӘжңҚеҠЎеҷЁйғҪжҳҜдёҖдёӘд»ЈзҗҶ(Broker). ж¶Ҳиҙ№иҖ…еҸҜд»Ҙи®ўйҳ…дёҖдёӘжҲ–еӨҡдёӘиҜқйўҳпјҢе№¶д»ҺBrokerжӢүж•°жҚ®пјҢд»ҺиҖҢж¶Ҳиҙ№иҝҷдәӣе·ІеҸ‘еёғзҡ„ж¶ҲжҒҜгҖӮ

еҗ¬иө·жқҘе’ҢJMSж¶ҲжҒҜеӨ„зҗҶе·®дёҚеӨҡпјҹ

и®©жҲ‘д»¬з«ҷзҡ„й«ҳдёҖзӮ№пјҢд»Һй«ҳзҡ„и§’еәҰжқҘзңӢпјҢKafkaйӣҶзҫӨзҡ„дёҡеҠЎеӨ„зҗҶе°ұеғҸиҝҷж ·еӯҗпјҡ

Clientе’ҢServerд№Ӣй—ҙзҡ„дәӨжөҒйҖҡиҝҮдёҖжқЎз®ҖеҚ•гҖҒй«ҳжҖ§иғҪе№¶дё”дёҚеұҖйҷҗжҹҗз§ҚејҖеҸ‘иҜӯиЁҖзҡ„TCPеҚҸи®®гҖӮйҷӨдәҶJava ClientеӨ–пјҢиҝҳжңүйқһеёёеӨҡзҡ„е…¶е®ғзј–зЁӢиҜӯиЁҖзҡ„ClientгҖӮ

иҜқйўҳе’Ңж—Ҙеҝ— (Topicе’ҢLog)

жӣҙж·ұе…Ҙзҡ„дәҶи§ЈдёҖдёӢKafkaдёӯзҡ„Topic.
TopicжҳҜеҸ‘еёғзҡ„ж¶ҲжҒҜзҡ„зұ»еҲ«жҲ–иҖ…з§ҚеӯҗFeedеҗҚгҖӮеҜ№дәҺжҜҸдёҖдёӘTopic, KafkaйӣҶзҫӨз»ҙжҠӨиҝҷдёҖдёӘеҲҶеҢәзҡ„log,е°ұеғҸдёӢеӣҫдёӯзҡ„зӨәдҫӢпјҡ

жӯЈеғҸдј з»ҹзҡ„ж¶ҲжҒҜзі»з»ҹдёҖж ·пјҢKafkaдҝқиҜҒж¶ҲжҒҜзҡ„йЎәеәҸдёҚеҸҳгҖӮ
еҶҚиҜҰз»ҶжүҜеҮ еҸҘгҖӮдј з»ҹзҡ„йҳҹеҲ—жЁЎеһӢдҝқжҢҒж¶ҲжҒҜпјҢе№¶дё”дҝқиҜҒе®ғд»¬зҡ„е…ҲеҗҺйЎәеәҸдёҚеҸҳгҖӮдҪҶжҳҜпјҢ е°Ҫз®ЎжңҚеҠЎеҷЁдҝқиҜҒдәҶж¶ҲжҒҜзҡ„йЎәеәҸпјҢж¶ҲжҒҜиҝҳжҳҜејӮжӯҘзҡ„еҸ‘йҖҒз»ҷеҗ„дёӘж¶Ҳиҙ№иҖ…пјҢж¶Ҳиҙ№иҖ…ж”¶еҲ°ж¶ҲжҒҜзҡ„е…ҲеҗҺйЎәеәҸдёҚиғҪдҝқиҜҒдәҶгҖӮиҝҷд№ҹж„Ҹе‘ізқҖе№¶иЎҢж¶Ҳиҙ№е°ҶдёҚиғҪдҝқиҜҒж¶ҲжҒҜзҡ„е…ҲеҗҺйЎәеәҸгҖӮз”ЁиҝҮдј з»ҹзҡ„ж¶ҲжҒҜзі»з»ҹзҡ„еҗҢеӯҰиӮҜе®ҡжё…жҘҡпјҢж¶ҲжҒҜзҡ„йЎәеәҸеӨ„зҗҶеҫҲи®©дәәеӨҙз—ӣгҖӮеҰӮжһңеҸӘи®©дёҖдёӘж¶Ҳиҙ№иҖ…еӨ„зҗҶж¶ҲжҒҜпјҢеҸҲиҝқиғҢдәҶе№¶иЎҢеӨ„зҗҶзҡ„еҲқиЎ·гҖӮ
еңЁиҝҷдёҖзӮ№дёҠKafkaеҒҡзҡ„жӣҙеҘҪпјҢе°Ҫз®Ўе№¶жІЎжңүе®Ңе…Ёи§ЈеҶідёҠиҝ°й—®йўҳгҖӮ KafkaйҮҮз”ЁдәҶдёҖз§ҚеҲҶиҖҢжІ»д№Ӣзҡ„зӯ–з•ҘпјҡеҲҶеҢәгҖӮ еӣ дёәTopicеҲҶеҢәдёӯж¶ҲжҒҜеҸӘиғҪз”ұж¶Ҳиҙ№иҖ…з»„дёӯзҡ„е”ҜдёҖдёҖдёӘж¶Ҳиҙ№иҖ…еӨ„зҗҶпјҢжүҖд»Ҙж¶ҲжҒҜиӮҜе®ҡжҳҜжҢүз…§е…ҲеҗҺйЎәеәҸиҝӣиЎҢеӨ„зҗҶзҡ„гҖӮдҪҶжҳҜе®ғд№ҹд»…д»…жҳҜдҝқиҜҒTopicзҡ„дёҖдёӘеҲҶеҢәйЎәеәҸеӨ„зҗҶпјҢдёҚиғҪдҝқиҜҒи·ЁеҲҶеҢәзҡ„ж¶ҲжҒҜе…ҲеҗҺеӨ„зҗҶйЎәеәҸгҖӮ
