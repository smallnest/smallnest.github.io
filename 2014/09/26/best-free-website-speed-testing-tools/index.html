Google PageSpeed Insights


йҰ–е…Ҳд»Ӣз»Қзҡ„иҝҷж¬ҫзҪ‘з«ҷйҖҹеәҰе’ҢжҖ§иғҪжөӢиҜ•е·Ҙе…·еҮәиҮӘ GoogleпјҢйҖҡиҝҮеҲҶжһҗзҪ‘йЎөеҶ…е®№жқҘдёәејҖеҸ‘иҖ…жҸҗдҫӣжҸҗеҚҮзҪ‘з«ҷеҠ иҪҪйҖҹзҡ„е»әи®®гҖӮPageSpeed зҡ„жөӢиҜ•еҢ…еҗ«з§»еҠЁи®ҫеӨҮе’ҢжЎҢйқўи®ҫеӨҮдёӨж–№йқўпјҢдҪҝжӮЁзҡ„зҪ‘йЎөеңЁжүҖжңүи®ҫеӨҮдёҠйғҪиғҪеҝ«йҖҹеҠ иҪҪгҖӮ

Yslow


YSlow жҳҜжңҖеёёз”Ёзҡ„зҪ‘з«ҷжҖ§иғҪеҲҶжһҗе·Ҙе…·д№ӢдёҖпјҢзӣёдҝЎ Web ејҖеҸ‘иҖ…зү№еҲ«жҳҜеүҚз«ҜејҖеҸ‘дәәе‘ҳдёҖе®ҡеҫҲзҶҹжӮүгҖӮYSlow еҹәдәҺдёҖз»„й«ҳжҖ§иғҪзҪ‘йЎөи§„еҲҷжқҘз»ҷдҪ жҸҗдҫӣе»әи®®пјҢеҰӮдҪ•дјҳеҢ–еҸҜд»Ҙи®©зҪ‘з«ҷжӣҙеҝ«гҖӮ

Pingdom


Pingdom жҳҜдёҖдёӘдјҹеӨ§зҡ„е·Ҙе…·пјҢеё®еҠ©з”ЁжҲ·зЎ®е®ҡзҪ‘з«ҷеҠ иҪҪж—¶й—ҙе№¶з”ҹжҲҗеӨ§йҮҸзҡ„жҠҘе‘ҠпјҢдҫӢеҰӮйЎөйқўеӨ§е°ҸгҖҒжөҸи§ҲеҷЁзј“еӯҳгҖҒжҖ§иғҪзӯүзә§зӯүгҖӮе®ғе…Ғи®ёжӮЁи·ҹиёӘжҖ§иғҪеҺҶеҸІе’Ңд»ҺдёҚеҗҢзҡ„ең°зҗҶдҪҚзҪ®иҝӣиЎҢжөӢиҜ•гҖӮеңЁзәҝжЈҖжҹҘзҪ‘з«ҷжҜҸдёӘе…ғзҙ зҡ„еҠ иҪҪйҖҹеәҰпјҢз”ҹжҲҗйқһеёёиҜҰз»Ҷзҡ„жөӢиҜ•жҠҘе‘ҠпјҢеё®еҠ©дҪ иҪ»жқҫдјҳеҢ–зҪ‘з«ҷгҖӮ

Load Impact


Load Impact её®еҠ©дҪ дёә Web еә”з”ЁзЁӢеәҸиҝӣиЎҢиҙҹиҪҪжөӢиҜ•е’ҢжҖ§иғҪжөӢиҜ•гҖӮйҖүжӢ©дёҖдёӘе…Ёзҗғзҡ„иҙҹиҚ·еҢәпјҢз„¶еҗҺжөӢиҜ•жЁЎжӢҹе®ўжҲ·гҖҒеёҰе®ҪгҖҒж•°жҚ®жҺҘеҸ—гҖҒжҜҸз§’зҡ„иҜ·жұӮзӯүзӯүгҖӮиҜҘе·Ҙе…·жҳҫзӨәдёҖдёӘжјӮдә®зҡ„еӣҫиЎЁжқҘжөӢйҮҸеҠ иҪҪж—¶й—ҙгҖӮ

Show Slow
