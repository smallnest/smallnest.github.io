жһҡдёҫзұ»еһӢд№ҹжҳҜеёёз”Ёзҡ„дёҖз§Қзұ»еһӢпјҢе®ғйҷҗе®ҡеӯ—ж®өзҡ„еҖјеҸӘиғҪеҸ–жҹҗдёӘзү№е®ҡзҡ„еҖјпјҢжҜ”еҰӮжҳҹжңҹзұ»еһӢеҸӘиғҪеҸ–е‘ЁдёҖеҲ°е‘Ёж—ҘдёғдёӘеҖјгҖӮ

еҰӮжһңиҰҒдҪҝз”Ёзҡ„зұ»еһӢеңЁе…¶е®ғprotoж–Үд»¶дёӯе®ҡд№үпјҢдҪ йңҖиҰҒдҪҝз”ЁimportжҠҠеҜ№еә”зҡ„ж–Үд»¶еј•е…ҘиҝӣжқҘгҖӮ

жңүж—¶еҖҷдҪ дёҚеҫ—дёҚдҝ®ж”№жӯЈеңЁдҪҝз”Ёзҡ„protoж–Үд»¶пјҢжҜ”еҰӮдёәзұ»еһӢеўһеҠ дёҖдёӘеӯ—ж®өпјҢprotobufж”ҜжҢҒиҝҷз§Қдҝ®ж”№иҖҢдёҚеҪұе“Қе·Іжңүзҡ„жңҚеҠЎпјҢдёҚиҝҮдҪ йңҖиҰҒйҒөеҫӘдёҖе®ҡзҡ„и§„еҲҷпјҡ

  • дёҚиҰҒж”№еҸҳе·Іжңүеӯ—ж®өзҡ„еӯ—ж®өзј–еҸ·
  • еҪ“дҪ еўһеҠ дёҖдёӘж–°зҡ„еӯ—ж®өзҡ„ж—¶еҖҷпјҢиҖҒзі»з»ҹеәҸеҲ—еҢ–еҗҺзҡ„ж•°жҚ®дҫқз„¶еҸҜд»Ҙиў«дҪ зҡ„ж–°зҡ„ж јејҸжүҖи§ЈжһҗпјҢеҸӘдёҚиҝҮдҪ йңҖиҰҒеӨ„зҗҶж–°еҠ еӯ—ж®өзҡ„зјәзңҒеҖјгҖӮ иҖҒзі»з»ҹд№ҹиғҪи§ЈжһҗдҪ дҝЎжҒҜзҡ„еҖјпјҢж–°еҠ еӯ—ж®өеҸӘдёҚиҝҮиў«дёўејғдәҶ
  • еӯ—ж®өд№ҹеҸҜд»Ҙиў«з§»йҷӨпјҢдҪҶжҳҜе»әи®®дҪ ReservedиҝҷдёӘеӯ—ж®өпјҢйҒҝе…Қе°ҶжқҘдјҡдҪҝз”ЁиҝҷдёӘеӯ—ж®ө
  • int32, uint32, int64, uint64 е’Ң boolзұ»еһӢйғҪжҳҜе…је®№зҡ„
  • sint32 е’Ң sint64е…је®№пјҢдҪҶжҳҜдёҚе’Ңе…¶е®ғж•ҙж•°зұ»еһӢе…је®№
  • string е’Ң bytesе…је®№пјҢеҰӮжһң bytes жҳҜеҗҲжі•зҡ„UTF-8 bytesзҡ„иҜқ
  • еөҢе…Ҙзұ»еһӢе’Ңbytesе…је®№пјҢеҰӮжһңbytesеҢ…еҗ«дёҖдёӘж¶ҲжҒҜзҡ„зј–з ҒзүҲжң¬зҡ„иҜқ
  • fixed32е’Ңsfixed32, fixed64е’Ңsfixed64
  • enumе’Ңint32, uint32, int64, uint64ж јејҸе…је®№
  • жҠҠеҚ•дёҖдёҖдёӘеҖјж”№еҸҳжҲҗдёҖдёӘж–°зҡ„oneofзұ»еһӢзҡ„дёҖдёӘжҲҗе‘ҳжҳҜе®үе…Ёе’ҢдәҢиҝӣеҲ¶е…је®№зҡ„гҖӮжҠҠдёҖз»„еӯ—ж®өеҸҳжҲҗдёҖдёӘж–°зҡ„oneofеӯ—ж®өд№ҹжҳҜе®үе…Ёзҡ„пјҢеҰӮжһңдҪ зЎ®дҝқиҝҷдёҖз»„еӯ—ж®өжңҖеӨҡеҸӘдјҡи®ҫзҪ®дёҖдёӘгҖӮжҠҠдёҖдёӘеӯ—ж®өз§»еҠЁеҲ°дёҖдёӘе·ІеӯҳеңЁзҡ„oneofеӯ—ж®өжҳҜдёҚе®үе…Ёзҡ„

е…¶е®ғ

еҰӮжһңжІЎжңүAnyзұ»еһӢпјҢеәҸеҲ—еҢ–еҗҺзҡ„ж•°жҚ®жҳҜжІЎжңүзұ»еһӢзҡ„е…ғж•°жҚ®дҝЎжҒҜзҡ„гҖӮиҝҷд№ҹж„Ҹе‘ізқҖеәҸеҲ—еҢ–зҡ„еҢ…е’ҢеәҸеҲ—еҢ–еҗҺзҡ„еҢ…еҸҜд»ҘдёҚдёҖж ·пјҢеҸӘиҰҒдҝқиҜҒж•°жҚ®ж јејҸзҡ„е®ҡд№үе…је®№еҚіеҸҜгҖӮ

иҝҳеҸҜд»ҘйҖҡиҝҮProtobufе®ҡд№үgrpcжңҚеҠЎпјҢдёҚиҝҮиҝҷдёӘж•ҷзЁӢдёҚж¶үеҸҠиҝҷдёҖж–№йқўзҡ„д»Ӣз»ҚгҖӮ

Well-Knownзұ»еһӢ

йҷӨдәҶжҲ‘д»¬еүҚйқўд»Ӣз»Қзҡ„еҹәжң¬зұ»еһӢгҖҒAnyгҖҒиҝҳжңүзңҒз•ҘжңӘд»Ӣз»Қзҡ„Serviceзӣёе…ізҡ„зұ»еһӢеӨ–пјҢиҝҳжңүдёҖдәӣеӨ§е®¶еёёи§Ғзҡ„зұ»еһӢпјҢProtobufд№ҹжҸҗдҫӣдәҶе®ҡд№үпјҢжҜ”еҰӮTimestampе’ҢDurationгҖӮиҝҷйҮҢжҲ‘д»¬зңӢзңӢеҰӮдҪ•дҪҝз”Ёе®ғгҖӮ

ProtobufжҸҗдҫӣдәҶgithub.com/golang/protobuf/ptypes/timestamp.Timestampе’Ңgithub.com/golang/protobuf/ptypes/duration.DurationдёӨз§Қжү©еұ•зұ»еһӢпјҢз”ЁжқҘиЎЁзӨәж—¶й—ҙпјҢе№¶жҸҗдҫӣдәҶе’Ңж ҮеҮҶеә“time.Timeе’Ң time.Durationзҡ„иҪ¬жҚўеҮҪж•°гҖӮ

дҪ еҸҜд»ҘеңЁдҪ зҡ„protoдёӯйңҖиҰҒж—¶й—ҙжҲіе’Ңdurationзҡ„ең°ж–№дҪҝз”ЁиҝҷдёӨдёӘзұ»еһӢпјҢиҖҢдёҚжҳҜдҪҝз”Ёж ҮеҮҶеә“зҡ„time.Timeе’Ңtime.Duration,еӣ дёәж ҮжіЁеә“зҡ„зұ»еһӢжІЎжңүжҸҗдҫӣprotobufеәҸеҲ—еҢ–еҸҚеәҸеҲ—еҢ–зҡ„еҠҹиғҪпјҢдҪ йңҖиҰҒйўқеӨ–еӨ„зҗҶпјҢжүҖд»ҘдёҚеҰӮзӣҙжҺҘдҪҝз”ЁProtobufжҸҗдҫӣзҡ„еҜ№еә”зұ»еһӢ(дёӢдёҖиҠӮgogoеә“жҸҗдҫӣдәҶеҜ№ж ҮеҮҶеә“зҡ„ж”ҜжҢҒпјҢзЁҚеҖҷеҶҚи®І)гҖӮ

еҗҢж—¶пјҢе®ғиҝҳжҳҜд»ҘжҢҮй’Ҳзҡ„ж–№ејҸе®ҡд№үеӯ—ж®өпјҢиҝҷд№ҹж„Ҹе‘ізқҖжҲ‘д»¬жҲ‘д»¬еҸҜд»ҘеҲҶеҲ«еҸҚеәҸеҲ—еҢ–зҡ„ж—¶еҖҷпјҢеҸҜд»ҘеҢәеҲҶеҜ№еә”еӯ—ж®өжҳҜеҗҰеңЁж•°жҚ®дёӯеӯҳеңЁгҖӮ

зј–з Ғ

йҰ–е…ҲпјҢжҲ‘д»¬е…ҲдәҶи§Јvarintж–№жі•гҖӮvarintж–№жі•жҳҜдёҖз§ҚдҪҝз”ЁеҸҳй•ҝж–№ејҸиЎЁзӨәж•ҙж•°зҡ„ж–№жі•пјҢеҸҜд»ҘдҪҝз”ЁдёҖдёӘжҲ–иҖ…еӨҡдёӘеӯ—иҠӮжқҘиЎЁзӨәе°Ҹж•ҙж•°е’ҢеӨ§ж•ҙж•°пјҢж•°и¶Ҡе°ҸпјҢдҪҝз”Ёзҡ„еӯ—иҠӮж•°и¶Ҡе°‘гҖӮ

еңЁvarintиЎЁзӨәзҡ„еӯ—иҠӮдёӯпјҢйҷӨдәҶжңҖеҗҺдёҖдёӘеӯ—иҠӮпјҢеүҚйқўзҡ„еӯ—иҠӮйғҪжңүдёҖдёӘbitжқҘиЎЁзӨәиҝҳжңүеӯ—иҠӮйңҖиҰҒеӨ„зҗҶпјҢиҝҷдёӘж Үи®°еҸ«еҒҡmost significant bit (msb) setгҖӮдҪҺдҪҚж”ҫеңЁеүҚйқўгҖӮ

жҜ”еҰӮ1иЎЁзӨәдёә0000 0001гҖӮжңҖй«ҳдҪҚ0иЎЁзӨәиҝҷжҳҜжңҖеҗҺдёҖдёӘеӯ—иҠӮдәҶпјҢеҸӘз”ЁдёҖдёӘеӯ—иҠӮе°ұеҸҜд»ҘиЎЁзӨәгҖӮ

ж•°еӯ—300иЎЁзӨәдёә1010 1100 0000 0010, дёӨдёӘеӯ—иҠӮжқҘиЎЁзӨәгҖӮжҜҸдёӘеӯ—иҠӮй«ҳдҪҚеҺ»жҺүеҚіеҸҜ: 010 1100 000 0010,еҸҚиҪ¬000 0010 010 1100,еҺ»жҺүеүҚйқўзҡ„0пјҢд№ҹе°ұжҳҜ100101100, 2^8 + 2^5 + 2^3 + 2^2= 256+32+8+4=300гҖӮ

Goж ҮеҮҶеә“encoding/binaryжңүеҜ№varintеӨ„зҗҶж–№жі•гҖӮ

дәӢе®һдёҠгҖӮProtobufжҳҜзј–з Ғзҡ„й”®еҖјеҜ№пјҢе…¶дёӯй”®з”ЁvarintжқҘиЎЁзӨәпјҢе…¶дёӯеҗҺдёүдҪҚд»ЈиЎЁwire typeгҖӮ

ProtobufеҸӘе®ҡд№үдәҶ6з§Қwireзұ»еһӢгҖӮ

еҜ№дәҺеӯ—ж®өжҜ”иҫғе°‘(2^4=16)жғ…еҶөпјҢдҪҝз”ЁдёҖдёӘеӯ—иҠӮе°ұеҸҜд»ҘиЎЁзӨәkeyгҖӮ

жҲ‘д»¬д»ҘдёҖдёӘз®ҖеҚ•зҡ„дҫӢеӯҗпјҢзңӢзңӢProtobufжҳҜеҰӮдҪ•иҝӣиЎҢзј–з Ғзҡ„гҖӮ

иҝҷдёӘдҫӢеӯҗзҡ„protoе®ҡд№үж–Үд»¶дёә:

е®ғе®ҡд№үдәҶPersionиҝҷдёӘmessageпјҢеҢ…еҗ«дёүдёӘеӯ—ж®өпјҢеҲҶеҲ«жҳҜstring, int32е’Ңstringзұ»еһӢпјҢе…¶дёӯз¬¬дёүдёӘеӯ—ж®өе…Ғи®ёйҮҚеӨҚгҖӮ

е®ҡд№үдёҖдёӘе®һдҫӢпјҡ

зј–з ҒеҗҺжҳҜдёҖдёӘ32еӯ—иҠӮзҡ„ж•°жҚ®пјҡ

з¬¬дёҖдёӘеӯ—ж®өзҡ„зұ»еһӢжҳҜеӯ—з¬ҰдёІ(wire typeжҳҜ2), еӯ—ж®өзј–еҸ·жҳҜ1 (00000001)пјҢ еӯ—ж®өзј–еҸ·е·Ұз§»дёүдҪҚеҶҚеҠ дёҠwiretypeе°ұжҳҜ0a(00001010)гҖӮ

з¬¬дәҢдёӘеӯ—ж®өзҡ„зұ»еһӢжҳҜint32(wire typeжҳҜ0), еӯ—ж®өзј–еҸ·жҳҜ2 (00000010)пјҢ еӯ—ж®өзј–еҸ·е·Ұз§»дёүдҪҚеҶҚеҠ дёҠwiretypeе°ұжҳҜ10(00010000)гҖӮ

з¬¬дёүдёӘеӯ—ж®өзҡ„зұ»еһӢжҳҜеӯ—з¬ҰдёІ(wire typeжҳҜ2), еӯ—ж®өзј–еҸ·жҳҜ3 (00000011)пјҢ еӯ—ж®өзј–еҸ·е·Ұз§»дёүдҪҚеҶҚеҠ дёҠwiretypeе°ұжҳҜ1a(00011010)гҖӮ

дёәдәҶжӣҙжңүж•Ҳзҡ„еӨ„зҗҶsint32е’Ңsint64, ProtobufйҮҮз”ЁZigZagзј–з ҒгҖӮ

еҜ№дәҺеӣәе®ҡй•ҝеәҰзҡ„ж•°еӯ—пјҢйҮҮз”Ёе°Ҹз«ҜеәҸзҡ„ж–№ејҸзј–з ҒпјҲlittle-endian byte orderпјүгҖӮ

еӯ—з¬ҰдёІеӨ„зҗҶkeyд№ӢеӨ–пјҢиҝҳйңҖиҰҒдёҖдёӘvarintи®°еҪ•е…¶й•ҝеәҰпјҢз„¶еҗҺжҳҜUTF-8зҡ„еӯ—з¬ҰдёІеҖјгҖӮ

еөҢе…Ҙзҡ„messageе’ҢbytesгҖҒstringдёҖж ·гҖӮ

Proto3дёӯеҜ№ж•°еӯ—зұ»еһӢзҡ„repeatedеӯ—ж®өйҮҮз”ЁpackеӨ„зҗҶж–№ејҸпјҢеҗҢдёҖдёӘrepeatedе…ғзҙ е…ұдә«еҗҢдёҖдёӘkeyпјҢд№ӢеҗҺжҳҜеӯ—ж®өзҡ„ж•ҙдҪ“еӯ—иҠӮй•ҝеәҰпјҢз„¶еҗҺжҳҜеҗ„дёӘе…ғзҙ гҖӮеӣ дёәж•°еӯ—зұ»еһӢеӨ©з”ҹе…·жңүеҸҜеҢәеҲҶжҖ§пјҢдёҚйңҖиҰҒйўқеӨ–зҡ„еҲҶйҡ”з¬ҰиҝӣиЎҢеҢәеҲҶгҖӮ

gogoеә“

иҷҪз„¶е®ҳж–№еә“golang/protobuжҸҗдҫӣдәҶеҜ№Protobufзҡ„ж”ҜжҢҒпјҢдҪҶжҳҜдҪҝз”ЁжңҖеӨҡиҝҳжҳҜз¬¬дёүж–№е®һзҺ°зҡ„еә“gogo/protobufгҖӮ

gogoеә“еҹәдәҺе®ҳж–№еә“ејҖеҸ‘пјҢеўһеҠ дәҶеҫҲеӨҡзҡ„еҠҹиғҪпјҢеҢ…жӢ¬пјҡ

  • еҝ«йҖҹзҡ„еәҸеҲ—еҢ–е’ҢеҸҚеәҸеҲ—еҢ–
  • жӣҙи§„иҢғзҡ„Goж•°жҚ®з»“жһ„
  • goprotobufе…је®№
  • еҸҜйҖүжӢ©зҡ„дә§з”ҹдёҖдәӣиҫ…еҠ©ж–№жі•пјҢеҮҸе°‘дҪҝз”Ёдёӯзҡ„д»Јз Ғиҫ“е…Ҙ
  • еҸҜд»ҘйҖүжӢ©дә§з”ҹжөӢиҜ•д»Јз Ғе’Ңbenchmarkд»Јз Ғ
  • е…¶е®ғеәҸеҲ—еҢ–ж јејҸ

жҜ”еҰӮetcdгҖҒk8sгҖҒdgraphгҖҒdocker swarmkitйғҪдҪҝз”Ёе®ғгҖӮ

еҹәдәҺйҖҹеәҰе’Ңе®ҡеҲ¶еҢ–зҡ„иҖғиҷ‘пјҢgogoжңүдёүз§Қдә§з”ҹд»Јз Ғзҡ„ж–№ејҸ

  • gofast: йҖҹеәҰдјҳе…ҲпјҢдёҚж”ҜжҢҒе…¶е®ғgogoprotobuf extensionsгҖӮ
  • gogofastгҖҒgogofasterгҖҒgogoslick: жӣҙеҝ«зҡ„йҖҹеәҰгҖҒжӣҙеӨҡзҡ„дә§з”ҹд»Јз Ғ

gogofastзұ»дјјgofast,дҪҶжҳҜдјҡеҜје…Ҙgogoprotobuf.
gogofasterзұ»дјјgogofast, дёҚдјҡдә§з”ҹXXX_unrecognizedжҢҮй’Ҳеӯ—ж®өпјҢеҸҜд»ҘеҮҸе°‘еһғеңҫеӣһж”¶ж—¶й—ҙгҖӮ
gogoslickзұ»дјјgogofaster,дҪҶжҳҜеҸҜд»ҘеўһеҠ дёҖдәӣйўқеӨ–зҡ„ж–№жі•gostringе’ҢequalзӯүзӯүгҖӮ

  • protoc-gen-gogo: жңҖеҝ«зҡ„йҖҹеәҰпјҢжңҖеӨҡзҡ„еҸҜе®ҡеҲ¶еҢ–

дҪ еҸҜд»ҘйҖҡиҝҮжү©еұ•е®ҡеҲ¶еәҸеҲ—еҢ–: жү©еұ•.
