ж„ҹзҹҘеҷЁз®—жі•дёҖдёӘзӣ‘зқЈеӯҰд№ зҡ„дәҢеҲҶеҲҶзұ»еҷЁпјҢ жҳҜзәҝжҖ§еҲҶзұ»еҷЁзҡ„дёҖз§ҚгҖӮ

ж„ҹзҹҘжңәз®—жі•жҳҜйқһеёёеҘҪзҡ„дәҢеҲҶзұ»з®—жі•пјҢиҜҘз®—жі•жұӮеҸ–дёҖдёӘеҲҶзҰ»и¶…е№ійқўпјҢи¶…е№ійқўз”ұwеҸӮж•°еҢ–е№¶з”ЁжқҘйў„жөӢпјҢеҜ№дәҺдёҖдёӘж ·жң¬xпјҢж„ҹзҹҘжңәз®—жі•йҖҡиҝҮи®Ўз®—y = [w,x]йў„жөӢж ·жң¬зҡ„ж ҮзӯҫпјҢжңҖз»Ҳзҡ„йў„жөӢж ҮзӯҫйҖҡиҝҮи®Ўз®—sign(y)жқҘе®һзҺ°гҖӮз®—жі•д»…еңЁйў„жөӢй”ҷиҜҜж—¶дҝ®жӯЈжқғеҖјwгҖӮ
е№іеқҮж„ҹзҹҘжңәе’Ңж„ҹзҹҘжңәз®—жі•зҡ„и®ӯз»ғж–№жі•дёҖж ·пјҢдёҚеҗҢзҡ„жҳҜжҜҸж¬Ўи®ӯз»ғж ·жң¬xiеҗҺпјҢдҝқз•ҷе…ҲеүҚи®ӯз»ғзҡ„жқғеҖјпјҢи®ӯз»ғз»“жқҹеҗҺе№іеқҮжүҖжңүжқғеҖјгҖӮжңҖз»Ҳз”Ёе№іеқҮжқғеҖјдҪңдёәжңҖз»ҲеҲӨеҲ«еҮҶеҲҷзҡ„жқғеҖјгҖӮеҸӮж•°е№іеқҮеҢ–еҸҜд»Ҙе…ӢжңҚз”ұдәҺеӯҰд№ йҖҹзҺҮиҝҮеӨ§жүҖеј•иө·зҡ„и®ӯз»ғиҝҮзЁӢдёӯеҮәзҺ°зҡ„йңҮиҚЎзҺ°иұЎгҖӮ
