tc: жЁЎжӢҹзҪ‘з»ңејӮеёёзҡ„е·Ҙе…·

Linux Traffic Control (tc)зҡ„жү©еұ• Network Emulation (netem)еҸҜд»ҘеҫҲж–№дҫҝзҡ„жЁЎжӢҹзҪ‘з»ңдёҚеҘҪзҡ„жғ…еҶөпјҢдёҖиҲ¬ж–°зҡ„linuxеҶ…ж ёдёӯ(>= 2.6)е·Із»ҸеҶ…зҪ®дәҶиҝҷдёӘе·Ҙе…·пјҢеҸҜд»Ҙж–№дҫҝзҡ„иҝӣиЎҢжөӢиҜ•гҖӮ

жң¬ж–ҮзҪ—еҲ—дәҶдәҶ tcзҡ„еёёз”Ёзҡ„жЁЎжӢҹе‘Ҫд»ӨпјҢ д»ҘеӨҮе°ҶжқҘдҪҝз”Ёзҡ„ж—¶еҖҷжҹҘиҜўгҖӮ

дё»иҰҒеҸӮиҖғдәҶLinuxеҹәйҮ‘дјҡзҡ„е®ҳж–№д»Ӣз»Қ: netem

зӣ‘жҺ§зҪ‘еҚЎгҖӮ

йҰ–е…ҲиҰҒжҹҘзңӢдҪ зҡ„зҪ‘еҚЎдҝЎжҒҜпјҢеҰӮ: eth0пјҢз„¶еҗҺе°ҶиҝҷдёӘзҪ‘еҚЎеҠ е…Ҙзӣ‘жҺ§еҲ—иЎЁ sudo tc qdisc add dev eth0 root netemгҖӮ

еҰӮжһңдёҚжғіеҶҚзӣ‘жҺ§пјҢеҸҜд»Ҙз§»йҷӨиҝҷдёӘзҪ‘еҚЎ sudo tc qdisc del dev eth0 root netem

еҰӮжһңжғіжҹҘзңӢзӣ‘жҺ§еҲ—иЎЁпјҢ еҸҜд»ҘдҪҝз”Ё tc -s qdiscгҖӮ
