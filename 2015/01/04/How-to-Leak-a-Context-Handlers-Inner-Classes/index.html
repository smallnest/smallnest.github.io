AndroidдёӯHandlerеј•иө·зҡ„еҶ…еӯҳжі„йңІ

еңЁAndroidеёёз”Ёзј–зЁӢдёӯпјҢHandlerеңЁиҝӣиЎҢејӮжӯҘж“ҚдҪңе№¶еӨ„зҗҶиҝ”еӣһз»“жһңж—¶з»Ҹеёёиў«дҪҝз”ЁгҖӮйҖҡеёёжҲ‘д»¬зҡ„д»Јз Ғдјҡиҝҷж ·е®һзҺ°гҖӮ

дҪҶжҳҜпјҢе…¶е®һдёҠйқўзҡ„д»Јз ҒеҸҜиғҪеҜјиҮҙеҶ…еӯҳжі„йңІпјҢеҪ“дҪ дҪҝз”ЁAndroid lintе·Ҙе…·зҡ„иҜқпјҢдјҡеҫ—еҲ°иҝҷж ·зҡ„иӯҰе‘Ҡ

In Android, Handler classes should be static or leaks might occur, Messages enqueued on the application threadвҖҷs MessageQueue also retain their target Handler. If the Handler is an inner class, its outer class will be retained as well. To avoid leaking the outer class, declare the Handler as a static nested class with a WeakReference to its outer class

зңӢеҲ°иҝҷйҮҢпјҢеҸҜиғҪиҝҳжҳҜжңүдёҖдәӣжҗһдёҚжё…жҘҡпјҢд»Јз Ғдёӯе“ӘйҮҢеҸҜиғҪеҜјиҮҙеҶ…еӯҳжі„йңІпјҢеҸҲжҳҜеҰӮдҪ•еҜјиҮҙеҶ…еӯҳжі„йңІзҡ„е‘ўпјҹйӮЈжҲ‘д»¬е°ұж…ўж…ўеҲҶжһҗдёҖдёӢгҖӮ

1) еҪ“дёҖдёӘAndroidеә”з”ЁеҗҜеҠЁзҡ„ж—¶еҖҷпјҢдјҡиҮӘеҠЁеҲӣе»әдёҖдёӘдҫӣеә”з”Ёдё»зәҝзЁӢдҪҝз”Ёзҡ„Looperе®һдҫӢгҖӮLooperзҡ„дё»иҰҒе·ҘдҪңе°ұжҳҜдёҖдёӘдёҖдёӘеӨ„зҗҶж¶ҲжҒҜйҳҹеҲ—дёӯзҡ„ж¶ҲжҒҜеҜ№иұЎгҖӮеңЁAndroidдёӯпјҢжүҖжңүAndroidжЎҶжһ¶зҡ„дәӢд»¶пјҲжҜ”еҰӮActivityзҡ„з”ҹе‘Ҫе‘Ёжңҹж–№жі•и°ғз”Ёе’ҢжҢүй’®зӮ№еҮ»зӯүпјүйғҪжҳҜж”ҫе…ҘеҲ°ж¶ҲжҒҜдёӯпјҢз„¶еҗҺеҠ е…ҘеҲ°LooperиҰҒеӨ„зҗҶзҡ„ж¶ҲжҒҜйҳҹеҲ—дёӯпјҢз”ұLooperиҙҹиҙЈдёҖжқЎдёҖжқЎең°иҝӣиЎҢеӨ„зҗҶгҖӮдё»зәҝзЁӢдёӯзҡ„Looperз”ҹе‘Ҫе‘Ёжңҹе’ҢеҪ“еүҚеә”з”ЁдёҖж ·й•ҝгҖӮ

2) еҪ“дёҖдёӘHandlerеңЁдё»зәҝзЁӢиҝӣиЎҢдәҶеҲқе§ӢеҢ–д№ӢеҗҺпјҢжҲ‘д»¬еҸ‘йҖҒдёҖдёӘtargetдёәиҝҷдёӘHandlerзҡ„ж¶ҲжҒҜеҲ°LooperеӨ„зҗҶзҡ„ж¶ҲжҒҜйҳҹеҲ—ж—¶пјҢе®һйҷ…дёҠе·Із»ҸеҸ‘йҖҒзҡ„ж¶ҲжҒҜе·Із»ҸеҢ…еҗ«дәҶдёҖдёӘHandlerе®һдҫӢзҡ„еј•з”ЁпјҢеҸӘжңүиҝҷж ·LooperеңЁеӨ„зҗҶеҲ°иҝҷжқЎж¶ҲжҒҜж—¶жүҚеҸҜд»Ҙи°ғз”ЁHandler#handleMessage(Message)е®ҢжҲҗж¶ҲжҒҜзҡ„жӯЈзЎ®еӨ„зҗҶгҖӮ

3) еңЁJavaдёӯпјҢйқһйқҷжҖҒзҡ„еҶ…йғЁзұ»е’ҢеҢҝеҗҚеҶ…йғЁзұ»йғҪдјҡйҡҗејҸең°жҢҒжңүе…¶еӨ–йғЁзұ»зҡ„еј•з”ЁгҖӮйқҷжҖҒзҡ„еҶ…йғЁзұ»дёҚдјҡжҢҒжңүеӨ–йғЁзұ»зҡ„еј•з”ЁгҖӮе…ідәҺиҝҷдёҖеҶ…е®№еҸҜд»ҘжҹҘзңӢз»ҶиҜқJavaпјҡвҖқеӨұж•ҲвҖқзҡ„privateдҝ®йҘ°з¬Ұ

зЎ®е®һдёҠйқўзҡ„д»Јз ҒзӨәдҫӢжңүзӮ№йҡҫд»ҘеҜҹи§үеҶ…еӯҳжі„йңІпјҢйӮЈд№ҲдёӢйқўзҡ„дҫӢеӯҗе°ұйқһеёёжҳҺжҳҫдәҶ

еҲҶжһҗдёҖдёӢдёҠйқўзҡ„д»Јз ҒпјҢеҪ“жҲ‘д»¬жү§иЎҢдәҶActivityзҡ„finishж–№жі•пјҢиў«е»¶иҝҹзҡ„ж¶ҲжҒҜдјҡеңЁиў«еӨ„зҗҶд№ӢеүҚеӯҳеңЁдәҺдё»зәҝзЁӢж¶ҲжҒҜйҳҹеҲ—дёӯ10еҲҶй’ҹпјҢиҖҢиҝҷдёӘж¶ҲжҒҜдёӯеҸҲеҢ…еҗ«дәҶHandlerзҡ„еј•з”ЁпјҢиҖҢHandlerжҳҜдёҖдёӘеҢҝеҗҚеҶ…йғЁзұ»зҡ„е®һдҫӢпјҢе…¶жҢҒжңүеӨ–йқўзҡ„SampleActivityзҡ„еј•з”ЁпјҢжүҖд»ҘиҝҷеҜјиҮҙдәҶSampleActivityж— жі•еӣһж”¶пјҢиҝӣиЎҢеҜјиҮҙSampleActivityжҢҒжңүзҡ„еҫҲеӨҡиө„жәҗйғҪж— жі•еӣһж”¶пјҢиҝҷе°ұжҳҜжҲ‘д»¬еёёиҜҙзҡ„еҶ…еӯҳжі„йңІгҖӮ

жіЁж„ҸдёҠйқўзҡ„new RunnableиҝҷйҮҢд№ҹжҳҜеҢҝеҗҚеҶ…йғЁзұ»е®һзҺ°зҡ„пјҢеҗҢж ·д№ҹдјҡжҢҒжңүSampleActivityзҡ„еј•з”ЁпјҢд№ҹдјҡйҳ»жӯўSampleActivityиў«еӣһж”¶гҖӮ

иҰҒи§ЈеҶіиҝҷз§Қй—®йўҳпјҢжҖқи·Ҝе°ұжҳҜдёҚйҖӮз”ЁйқһйқҷжҖҒеҶ…йғЁзұ»пјҢз»§жүҝHandlerж—¶пјҢиҰҒд№ҲжҳҜж”ҫеңЁеҚ•зӢ¬зҡ„зұ»ж–Үд»¶дёӯпјҢиҰҒд№Ҳе°ұжҳҜдҪҝз”ЁйқҷжҖҒеҶ…йғЁзұ»гҖӮеӣ дёәйқҷжҖҒзҡ„еҶ…йғЁзұ»дёҚдјҡжҢҒжңүеӨ–йғЁзұ»зҡ„еј•з”ЁпјҢжүҖд»ҘдёҚдјҡеҜјиҮҙеӨ–йғЁзұ»е®һдҫӢзҡ„еҶ…еӯҳжі„йңІгҖӮеҪ“дҪ йңҖиҰҒеңЁйқҷжҖҒеҶ…йғЁзұ»дёӯи°ғз”ЁеӨ–йғЁзҡ„Activityж—¶пјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёејұеј•з”ЁжқҘеӨ„зҗҶгҖӮеҸҰеӨ–е…ідәҺеҗҢж ·д№ҹйңҖиҰҒе°ҶRunnableи®ҫзҪ®дёәйқҷжҖҒзҡ„жҲҗе‘ҳеұһжҖ§гҖӮжіЁж„ҸпјҡдёҖдёӘйқҷжҖҒзҡ„еҢҝеҗҚеҶ…йғЁзұ»е®һдҫӢдёҚдјҡжҢҒжңүеӨ–йғЁзұ»зҡ„еј•з”ЁгҖӮ дҝ®ж”№еҗҺдёҚдјҡеҜјиҮҙеҶ…еӯҳжі„йңІзҡ„д»Јз ҒеҰӮдёӢ

е…¶е®һеңЁAndroidдёӯеҫҲеӨҡзҡ„еҶ…еӯҳжі„йңІйғҪжҳҜз”ұдәҺеңЁActivityдёӯдҪҝз”ЁдәҶйқһйқҷжҖҒеҶ…йғЁзұ»еҜјиҮҙзҡ„пјҢе°ұеғҸжң¬ж–ҮжҸҗеҲ°зҡ„дёҖж ·пјҢжүҖд»ҘеҪ“жҲ‘д»¬дҪҝз”Ёж—¶иҰҒйқһйқҷжҖҒеҶ…йғЁзұ»ж—¶иҰҒж јеӨ–жіЁж„ҸпјҢеҰӮжһңе…¶е®һдҫӢзҡ„жҢҒжңүеҜ№иұЎзҡ„з”ҹе‘Ҫе‘ЁжңҹеӨ§дәҺе…¶еӨ–йғЁзұ»еҜ№иұЎпјҢйӮЈд№Ҳе°ұжңүеҸҜиғҪеҜјиҮҙеҶ…еӯҳжі„йңІгҖӮдёӘдәәеҖҫеҗ‘дәҺдҪҝз”Ёж–Үз« зҡ„йқҷжҖҒзұ»е’Ңејұеј•з”Ёзҡ„ж–№жі•и§ЈеҶіиҝҷз§Қй—®йўҳгҖӮ
