  • Copy：支持 Copy 的类型可以安全地通过字节复制的方式进行复制，可以类比 C 语言中的 memcpy 函数。

  • Clone：支持 Clone 的类型也可以被复制，但它通常需要执行一些逻辑操作来完成深拷贝。
